### NGINX
- **一键启动NGINX服务**: 利用nodejs配置nginx的conf实现**一键启动NGINX**和**NGINX反向代理**配置
- 启动：`npm run nginx`
- 访问：浏览器打开http://localhost:8017/index.html或者http://localhost:8016/index.html
- 停止：`npm run stop`
